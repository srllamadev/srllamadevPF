# Personal-Web-Site

Este repositorio contiene un sitio web estático tipo "Portfolio" para Yamil Navia (srllamadev).

Archivos principales:

- `index.html` — Página principal (estructura bento, secciones separadas).
- `styles.css` — Estilos (dark futuristic bento).
- `script.js` — Interacciones mínimas (copiar email, año dinámico).
- `assets/img/profile.jpg` — Foto de perfil.
- `assets/img/profile2.jpg` — Marca/llama.

Para ver el sitio localmente, abre `index.html` en tu navegador (doble clic) o sirve la carpeta con un servidor simple.

Ejemplo con Python (PowerShell / Windows):

```powershell
python -m http.server 8000
# luego abrir http://localhost:8000 en el navegador
```
